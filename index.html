IFNL-Evolution

Comparative evolutionary analysis of IFNL genes

Download .zip Download .tar.gz View on GitHub

Description

This GitHub repository contains data and program logic supporting comparative and phylogenetic investigations of interferon lambda IFNL evolution.

Nomenclature

IFNL is a cytokine and is also referred to as Interleukin-28 (IL-28).

Sequence data

The sequence data in this project have been organised into the following sources:

ncbi-refseqs: mRNA reference sequences of IFNL genes from distinct species. These XML-formatted files are downloaded directly from NCBI using a GLUE module (see here) and are uniquely identified within this project by their GenBank accession numbers.

fasta-curated: A non-redundant set of loci disclosing similarity to IFNL proteins. These FASTA sequences have been curated via systematic screening of whole genome sequence (WGS) assemblies using the DIGS tool. Sequences in this source have unique IDs based on arbitrary numbering.
