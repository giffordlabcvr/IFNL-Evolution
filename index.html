IFNL-Evolution

Comparative evolutionary analysis of IFNL genes

Download .zip Download .tar.gz View on GitHub

Description

This GitHub repository contains data and program logic supporting comparative and phylogenetic investigations of interferon lambda IFNL evolution.

IFNL is a cytokine and is also referred to as Interleukin-28 (IL-28).
